#Комманды для запуска проекта  
*Первый запуск*  

**make first_start**
- Применяет миграции
- Загружает данные из sqlite в postgresql
- Тестирует загруженные данные
- Добавляет суперюзера указанного в .env файле
- Удаляет не нужные более папки и файлы
- Собирает статику

Все комманды make  

**make help**
